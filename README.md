你好呀，欢迎来到我的 GitHub 主页。

我是 **Anderwer**，一名喜欢折腾代码的学生。  
平时主要关注：

- 🧩 **后端开发**：喜欢思考接口设计、数据处理、服务逻辑和工程结构
- 🤖 **AI / 算法**：对机器学习、智能应用、算法题和模型落地很感兴趣
- 🐍 **Python 生态**：日常写得最多的语言，拿来做后端、脚本、数据处理都很顺手

## 🪄 我的理想项目风格

我比较喜欢这类项目：

- 功能明确，结构清晰
- 能真正解决一个小问题
- 不只是“能跑”，还尽量“像一个完整作品”
- 最好带一点技术探索属性，例如：
  - 后端服务设计
  - 模型调用 / 推理接口
  - 自动化脚本
  - 数据分析与处理
  - 算法可视化 / 练习总结
